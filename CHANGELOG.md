# Changelog

All notable changes to this project will be documented in this file.
Format based on [Keep a Changelog](https://keepachangelog.com/).

## [0.2.0] - 2026-04-30

### Added
- 定时自动化：每天凌晨 3 点采集并总结前一天事项，再基于昨天总结生成当天计划。
- 当天刷新任务：每 4 小时刷新当天“干了啥”和“总结下”，并通过后台任务队列去重。
- 异步后台任务入口与最近任务面板，覆盖总结生成、计划生成、图谱刷新/重建、每日流水线和当天刷新。
- 总结结构化输出：工作日程、项目进展/问题/风险/下一步、知识沉淀和时间分布均返回结构化字段。
- 知识图谱实体归一化层，统一“我/user/本人”等自指实体，并处理泛化角色与别名合并。
- “规划下”支持在生成计划下方手动新增计划项，并编辑优先级、状态、时长和建议时段。

### Changed
- “干了啥”“总结下”“规划下”的手动触发入口降级为备用操作，主流程转向定时自动化。
- “总结下”页面改为竖向时间轴、项目分组进展、知识卡片和条形时间分布。
- “整理下”页面改为年份 + 季度、月份、日期的多级历史结构。
- “沉淀下”知识图谱切换为更有物理感的力导向展示。
- 配置页统一数据源卡片和标题样式，Chrome、Safari、Google Calendar、Gmail 使用对应图标。
- macOS App 重新打包流程修正缓存和运行目录行为，确保看到最新前端 UI。

## [0.1.0] - 2026-04-03

### Added
- 数据导入方式：Chrome 浏览历史 JSON 上传、浏览器本地采集、Google Calendar 采集、手动输入活动记录（AC-1）
- AI 意图理解：自动推断活动类别（工作/学习/生活/娱乐）、意图和主题标签（AC-2）
- 每日总结生成：时间线动线、事项进展、新知识提取、时间分布统计（AC-3）
- 轻量知识图谱：基于 SQLite 的增量式知识图谱，自动提取实体和关系（AC-4）
- 计划与建议：AI 生成带优先级的明日计划和注意力/学习优化建议，可编辑（AC-5）
- LLM 双模式支持：OpenAI API 和 Ollama 本地模型配置切换（AC-6）
- 数据管理：按日期浏览数据概览，支持单条和整天数据物理删除（AC-7）
- 后端 API：FastAPI + SQLAlchemy + SQLite，16 个 RESTful 接口
- 测试覆盖：53 个单元测试 + 8 个 E2E 测试场景，通过率 100%
